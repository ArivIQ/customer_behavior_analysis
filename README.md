# customer_behavior_analysis
# 📊 Data Analytics Project

## Overview

This project demonstrates an end-to-end data analytics workflow, starting from raw data processing to interactive dashboard creation and business reporting. The project covers data loading, exploratory data analysis (EDA), data cleaning, SQL analysis using PostgreSQL, dashboard development in Power BI, and presentation of insights.

---

## Dataset

The dataset contains structured business data used for analysis and visualization.

The analysis includes:

* Data inspection
* Missing value handling
* Duplicate removal
* Data type correction
* Exploratory Data Analysis (EDA)
* Business insights

## Project Workflow

### 1. Data Loading

* Imported the dataset into Python using Pandas.
* Performed an initial inspection of the data.

### 2. Exploratory Data Analysis (EDA)

* Examined data distributions.
* Identified missing values and duplicates.
* Generated summary statistics.
* Created visualizations to understand trends.

### 3. Data Cleaning

* Removed duplicate records.
* Handled missing values.
* Corrected data types.
* Prepared the dataset for analysis.

### 4. SQL Analysis (PostgreSQL)

* Loaded the cleaned dataset into PostgreSQL.
* Executed SQL queries to answer business questions.
* Used filtering, grouping, joins, aggregate functions, and window functions for analysis.

### 5. Power BI Dashboard

Created an interactive dashboard featuring:

* KPI Cards
* Bar Charts
* Line Charts
* Pie/Donut Charts
* Filters and Slicers
* Business performance insights

### 6. Report & Presentation

* Prepared a detailed analytics report.
* Created a presentation using Gamma.
* Summarized findings with visuals and recommendations.

## How to Run

1. Clone this repository.
2. Install the required Python libraries:

   ```bash
   pip install pandas numpy matplotlib
   ```
3. Open the Jupyter Notebook files and run them in sequence.
4. Import the cleaned dataset into PostgreSQL.
5. Execute the SQL queries.
6. Open the Power BI dashboard (`.pbix`) file.
7. Review the project report and presentation for detailed insights.
